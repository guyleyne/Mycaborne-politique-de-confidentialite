## Présentation

Ce repository héberge les documents légaux de MyCarbone, extension de mesure d'empreinte carbone pour requêtes IA, conçue selon le principe de Privacy by Design.

**Objet des documents :**
- Informer les utilisateurs sur le traitement de leurs données
- Répondre aux exigences réglementaires (RGPD)
- Satisfaire aux conditions des places de marché d'extensions
- Garantir la transparence totale sur l'architecture technique

## Contenu de la politique

La politique de confidentialité détaille les points suivants :

### 1. Principes fondamentaux
- Architecture 100% locale
- Absence de collecte de données personnelles
- Aucune communication avec des serveurs externes

### 2. Données traitées
- Ce qui n'est PAS collecté (identité, historique, conversations)
- Ce qui est stocké localement (statistiques agrégées, budget)

### 3. Base légale
- Qualification au regard du RGPD
- Traitements exclusivement locaux

### 4. Droits des utilisateurs
- Contrôle total via l'interface
- Suppression immédiate possible
- Portabilité des données

### 5. Contact
- Équipe juridique et support
